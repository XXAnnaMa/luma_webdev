## 📖 About

A modern, feature-rich social platform for discovering and sharing local events. Built with React and Tailwind CSS, this application provides an intuitive interface for users to explore activities, connect with like-minded people, and build their social footprint.

## 🚀 Installation

### Prerequisites

- Node.js 18+ 
- pnpm (recommended) or npm

### Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/event-social-platform.git
   cd event-social-platform
   ```

2. **Install dependencies**
   ```bash
   pnpm install
   ```

3. **Start development server**
   ```bash
   pnpm dev
   ```

4. **Open your browser**
   ```
   Navigate to http://localhost:5173
   ```

### Build for Production

```bash
pnpm build
```
